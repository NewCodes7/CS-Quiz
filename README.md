## 기술 스택

### 사용 기술

- Java 17
- SpringBoot `3.2.5`
- gradle `8.7`
- thymeleaf
- JDBC template
- 데이터베이스 : MySQL

### 기타 사용 기술

- JUnit5
- 필터(Spring Interceptor)
    - 세션 쿠키를 이용하여 로그인 상태인지 확인
    - 로그인상태가 아닐 시 게시판 메인 화면, 회원가입, 로그인 페이지 등 접근, 기능 제한
